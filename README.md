coca
===================================
Co-Cache 缓存的协同框架

## 运行要求
- Jdk8

## 工程说明

## 快速开始
- 示例说明
    - 同步两个coca实例的本地缓存改动
    - 需要安装redis服务器
    
- 添加Maven依赖

```xml
<dependency>
    <groupId>dzh.github.io</groupId>
    <artifactId>coca-example</artifactId>
    <version>1.0.0</version>
</dependency>
<dependency>
    <groupId>dzh.github.io</groupId>
    <artifactId>coca-redis</artifactId>
    <version>1.0.0</version>
</dependency>
<dependency>
    <groupId>dzh.github.io</groupId>
    <artifactId>coca-ehcache</artifactId>
    <version>1.0.0</version>
</dependency>
```
