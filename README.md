coca
===================================
Co-Cache 缓存的协同框架


## 快速开始
- 示例说明
    - 同步两个coca实例的本地缓存改动
    - 需要安装redis服务器
    
- 添加Maven依赖

```xml
<dependency>
    <groupId>dzh.github.io</groupId>
    <artifactId>coca-api</artifactId>
</dependency>
<dependency>
    <groupId>dzh.github.io</groupId>
    <artifactId>coca-redis</artifactId>
</dependency>
<dependency>
    <groupId>dzh.github.io</groupId>
    <artifactId>coca-guava</artifactId>
    <version>1.0.0</version>
</dependency>
```

## 运行要求
- (0.0.1, 1.0.0)
    - jdk7 or higher 
- [1.0.0, 2.0.0)
    - jdk8
- [2.0.0, +∞)
    - jdk9
